# 知识库文档目录

此目录用于存放角色设定、背景故事等知识库文档。

## 📝 使用说明

### 1. 准备文档

将你的角色设定文档放入此目录，支持以下格式：
- `.txt` - 纯文本文件
- `.md` - Markdown 文件

### 2. 文档内容建议

为了让机器人更好地理解角色，建议包含以下内容：

#### 基本信息
- 角色姓名、年龄、身份
- 外貌特征
- 性格特点

#### 背景故事
- 角色经历
- 重要事件
- 成长历程

#### 人际关系
- 与其他角色的关系
- 重要的人物
- 情感纠葛

#### 行为特征
- 说话方式
- 口头禅
- 行为习惯

#### 世界观设定
- 所处的世界背景
- 相关设定
- 特殊能力或技能

### 3. 构建知识库

准备好文档后，运行知识库构建工具：

**Windows:**
```bash
# 双击运行
知识库构建工具.bat

# 或在命令行运行
python tools/rebuild_knowledge_base.py
```

**Linux/macOS:**
```bash
python tools/rebuild_knowledge_base.py
```

### 4. 文档示例

#### 示例文件名
- `月代雪设定集.txt`
- `角色背景.md`
- `世界观设定.txt`
- `人物关系.md`

#### 示例内容结构

```markdown
# 月代雪 角色设定

## 基本信息
- 姓名：月代雪（Tsukishiro Yuki）
- 年龄：18岁
- 身份：大魔女，魔女种族最后的幸存者

## 性格特征
- 冷静琉璃
- 很少安慰他人
- 内心背负着沉重的仇恨

## 背景故事
月代雪是魔女种族最后的幸存者...

## 人际关系
- 艾玛：特殊感情
- 希罗：特殊感情

## 说话风格
- 语气平淡
- 简短直接
- 很少使用感叹号
```

## ⚠️ 注意事项

1. **版权问题**
   - 请确保你有权使用这些设定内容
   - 不要上传侵犯他人版权的内容
   - 本项目仅供学习交流使用

2. **文件编码**
   - 建议使用 UTF-8 编码
   - 避免使用特殊字符

3. **文件大小**
   - 单个文件建议不超过 10MB
   - 总文档数量建议不超过 100 个

4. **更新知识库**
   - 每次修改文档后需要重新构建知识库
   - 构建过程可能需要几分钟时间

## 🔧 故障排除

### 构建失败
- 检查文档编码是否为 UTF-8
- 确认文档格式是否正确
- 查看日志文件了解详细错误

### 知识库不生效
- 确认已重新构建知识库
- 检查 `data/` 目录下是否生成了相关文件
- 重启机器人
